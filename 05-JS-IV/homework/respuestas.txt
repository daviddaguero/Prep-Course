- Objeto: es una estructura dentro de un programa que tiene una serie de datos almacenados y tareas que realiza con esos datos, las cuales son conocidas como métodos.
- Propiedades: pueden ser datos o métodos que están almacenados dentro del objeto. Los datos pueden ser de cualquier tipo, incluso se pueden agregar objetos como propiedades de un objeto. Se almacenan de la forma key-value que significa que hay una palabra (key) que sirve para acceder a un valor determinado.
- Métodos: son las funciones que puede tener como propiedad un objeto. Las funciones se encargan de hacer operaciones, generalmente usando los datos que están almacenados en las otras propiedades del objeto.
- Bucle for...in: es un statement que nos permite iterar sobre las propiedades de un objeto. Este bucle recorre todo el objeto y finaliza cuando pasó por todas las propiedades del mismo.
- Notación de puntos vs notación de corchetes: la notación de puntos es una forma de acceder a propiedades del objeto de una manera más fácil. Basta con escribir el nombre del objeto, seguido de un punto y la llave de la propiedad a la que queremos acceder. La notación de puntos es literal, lo que significa que no la podemos usar con expresiones, sino solo con el nombre de la propiedad buscada. La notación de corchetes nos permite hacer lo mismo pero usando el nombre de la propiedad entre corchetes y comillas. A través de esta última notación, sí podemos usar expresiones para acceder a los elementos del objeto. Para usar expresiones tenemos que poner el nombre del objeto seguido de corchetes y la expresión entre ellos.
